# Data Structures & Algorithms (DSA) - CRUD & LeetCode

This repository contains my personal implementations of key Data Structures and Algorithms (DSA) in Python, designed around core **CRUD (Create, Read, Update, Delete)** operations, along with approaches to corresponding **LeetCode** problems.

## 📁 Repository Structure

*   **`CRUD/`**: From-scratch implementations of linear and non-linear data structures.
    *   `linkedlist/`: Singly, Doubly, Circular, and Circular Doubly Linked Lists.
    *   `Stack/`: Stack implementations using lists, linked lists, composition, and inheritance.
    *   `queues/`: Queue implementations using lists and linked lists.
    *   `dequeue/`: Double-ended queues.
    *   `priority queue/`: Priority queues using lists and linked lists.
    *   `Tree/`: Hierarchical structures like Binary Search Trees (BST).
    *   `pythonbasics/`: Python fundamentals and OOP concepts.

---

## 🛠 Features covered (Linear & Non-Linear)

### 1. Linear Data Structures
*   **Linked Lists:** Custom pointer structures with full insertion, search, traversal, and deletion.
*   **Stacks:** LIFO operations.
*   **Queues & Dequeues:** FIFO and Double-ended insertion/removal.
*   **Priority Queues:** Elements sorted based on priority.

### 2. Non-Linear Data Structures
*   **Binary Search Tree (BST):** Nodes structured hierarchically with recursion, traversal, and dynamic manipulation.

### 3. LeetCode Problems (Upcoming)
*   Solutions categorized by data structure and topic, detailing brute force vs. optimized approaches.
